TYPO3.Extbase Extension zur Verwendung des Bootstrap 3 Carousel 
==============================================================

Als Grundlage dient das Bootstrap 3 Carousel:
http://getbootstrap.com/javascript/#carousel

* Extension installieren
* Extension als statisches Template einbinden 
* Datensätze im Backend anlegen und ID der Datensatzsammlung in den Konstanten der Extension eintragen
* PlugIns auf der gewünschten Seite eintragen
* Viel Spaß
